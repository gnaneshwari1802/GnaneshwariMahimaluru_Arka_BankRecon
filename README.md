## Repository Overview

This repository contains my submission for the **DeepThought (DT) Finance Fellowship – Round 1** Bank Reconciliation Assignment.

The assignment involved reconciling the HDFC Bank Statement with the Zoho Books Ledger for **Arka Tea Traders Pvt. Ltd.** for the period:

**01 January 2026 – 31 March 2026**

The objective was to identify discrepancies, prepare a formal Bank Reconciliation Statement (BRS), recommend correcting journal entries, analyze the root causes, and propose process improvements.

---

# Assignment Objectives

The assignment required:

- Compare the Zoho Books ledger with the HDFC Bank statement.
- Identify all discrepancies.
- Classify discrepancies into the prescribed categories.
- Prepare a formal Bank Reconciliation Statement.
- Record correcting journal entries wherever required.
- Perform root cause analysis.
- Write a professional memo to the Finance Head.
- Document my complete problem-solving approach through a hand-drawn thought process.
